### 고민사항

-   오픈소스 디자인 템플릿을 활용 or 하드코딩
-   Layout을 별도로 분리 or page 내부에 종속
-   common 디렉토리 생성 여부
-   위와 다르게 아토믹 디자인에 의거하여 template추가하여 작업할지 고민
